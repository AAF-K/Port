Project 01: Raspberry Pi

Raspberry Pi Smart Home Server

Een complete zelfgebouwde server voor cloud storage, home automation en lokale AI

Project Overview

Dit project begon met een simpele vraag: kan ik mijn eigen cloud storage bouwen die even goed werkt als Google Drive of Icloud, maar dan volledig onder mijn eigen controle? Het antwoord werd veel groter dan alleen cloud storage.

Ik heb een Raspberry Pi 5 omgetoverd tot een volwaardige smart home server die drie belangrijke functies vervult: veilige cloud storage voor al mijn bestanden, een centraal platform voor al mijn smart home apparaten via Home Assistant, en een lokaal draaiend Large Language Model voor privacy-vriendelijke AI assistentie.

Layer 5: External Access
DuckDNS Dynamic DNS + VPN (WireGuard)

Uitdagingen & Oplossingen

Technische Uitdagingen

  • Limited Resources: Raspberry Pi heeft beperkte RAM. Oplossing: Docker memory limits en swap optimalisatie.
  • Network Setup: Externe toegang via CGNAT router. Oplossing: WireGuard VPN + DuckDNS voor dynamische DNS.
  • LLM Performance: Grote modellen draaien traag. Oplossing: Kleinere, geoptimaliseerde modellen (Phi-2, TinyLlama).
  • Storage Management: Bestandssync conflicten. Oplossing: Nextcloud versioning + conflict resolution scripts.
  • Uptime: Server moet 24/7 beschikbaar zijn. Oplossing: Automatische restart scripts + UPS battery backup.
